# Protocol "Wilds" (Godot Project)

## Обзор
Двумерный roguelike-шутер с элементами выживания и строительства, разработанный на Godot 4.5.

## Запуск
1. Откройте проект в Godot Engine (версия 4.5+).
2. Нажмите **F5** для запуска основной сцены.
